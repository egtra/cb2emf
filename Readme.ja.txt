CB2EMF 0.1
========
クリップボードにあるEMFデータを取り出して、EMFファイルにして保存します。

Web site: https://github.com/egtra/cb2emf

使用法
----------------
1. 適当なアプリケーションでコピーしてください。
（作者はMicrosoft Excelのグラフで動くことを確かめています）
2. コマンドプロンプトでこんな風に打ち込んでください。
    cb2emf output.emf

注意
----------------
そもそも、クリップボードにEMFまたはWMFでコピーしてくれない場合は
対応できません。あくまで、「クリップボードにあるEMFデータを取り出したい」
という目的で作成したアプリケーションですので。
